# 🐢 Turtle Assistant

A responsive and interactive turtle assistant web application built with modern web technologies.

## 📋 Features

- 🐢 Interactive turtle assistant interface
- 🎨 Beautiful and responsive design
- ⚡ Fast performance with Vite
- 📱 Mobile-friendly UI
- 🚀 Easy deployment to Netlify

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ashwinverma00-del/https-turtle-assitant-ff11d2.netlify.app.git
   cd https-turtle-assitant-ff11d2.netlify.app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your configuration
   ```

### Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:3000`

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build locally:

```bash
npm run preview
```

## 🌐 Deployment

### Deploy to Netlify

1. Push your code to GitHub
2. Connect your repository to Netlify
3. Set the build command to `npm run build`
4. Set the publish directory to `dist`
5. Deploy!

Or use Netlify CLI:

```bash
npm install -g netlify-cli
netlify deploy
```
